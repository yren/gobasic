## short cut
* cmd + 1, 显示项目结构
* cmd + 4, 显示 run window, 运行结果

* live template, "main"
* cmd + j, 显示所有 live template
* "sout" , system.out. live template
* ctrl + space, 编码提示

* opt + cmd + l, format code
* shift + opt + cmd + l, show format code dialog

* create test, 在类名， opt + enter, 选择 create test
