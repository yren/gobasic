

## productivity killer

* cmd + shift + a, Actions, Tab placement: none (on)
* cmd + e, navigate recent files
* shift + cmd + e, navigate recent edit files

## move cursor

* opt + -> , move a word
* cmd + -> , move to line end (begin)
* fn + up | down, move a page
* fn + cmd + ->, move to file end (begin)
